== Installation ==

*Server Requirements:* PHP4 ou PHP5.

*Wordpress versions:* Wordpress 2.7 et supérieur.

Note: Vous devez [créer un compte gratuitement sur Notyf](https://notyf.com/register) pour utiliser Notyf.

Guide d'utilisation :

* Installer l'extension Wordpress et l'activer.
* Dans le menu admin, cliquer sur “Notyf” et copier-coller le code d'installation.
* Toutes les modifications se passent directement depuis [votre compte Notyf](https://notyf.com/register).
* Et voilà, vos visiteurs voient apparaître des preuves sociales et des notifications percutantes.
